# dujiaokauser
独角发卡2.0.6二开用户版，只适配hyper模板
交流群：https://t.me/dujiaouser


基本环境要求
(PHP + PHPCLI) version = 7.4
Nginx version >= 1.16
MYSQL version >= 5.6
Redis (高性能缓存服务)
Supervisor (一个python编写的进程管理服务)
Composer (PHP包管理器)
Linux (Win下未测试，建议直接Linux)
PHP环境要求
星号(*)为必须执行的要求，其他为建议内容

*安装fileinfo扩展
*安装redis扩展
*终端需支持php-cli，测试php -v(版本必须一致)
*需要开启的函数：putenv，proc_open，pcntl_signal，pcntl_alarm
安装opcache扩展
默认后台
后台路径 /admin
默认管理员账号 admin
默认管理员密码 admin
